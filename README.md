# zh-m-notes

本仓库用于归档和发布多个独立 HTML 页面。项目不针对任何特定域名或网站，仅作为静态页面的存储与展示空间。

## 项目简介

这是一个轻量级的 HTML 页面集合仓库。每个页面独立存在，内容涉及个人笔记、技术演示、工具页面等。仓库旨在提供一个整洁的页面归档环境，方便随时访问和更新。

## 目录说明

```
├── index.html          # 页面入口/导航页
├── pages/              # 主要页面存放目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `index.html`：仓库入口页面，通常包含页面列表或导航。
- `pages/`：存放所有独立 HTML 页面，按需命名。
- `assets/`：存放各页面共用的样式、脚本或图片资源。

## 页面归档说明

- 每个页面均为独立 HTML 文件，可单独打开使用。
- 页面内容不依赖外部服务，尽量保持自包含。
- 页面命名遵循简洁、有意义的规则，便于识别。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进现有页面或新增内容。
- 新页面请放入 `pages/` 目录，并在 `index.html` 中添加链接。
- 保持页面结构清晰，避免引入恶意代码或外部依赖。
- 本仓库为个人归档用途，不保证内容长期可用或更新。
